游民星空 club圈子 1.0 版源文件
---
- project/ 为项目文件

默认安装[node](https://nodejs.org/en/),[cnpm](https://npm.taobao.org/)。  

开发：
1. cnpm install
2. gulp dev  

打包：  
1. gulp pub
> 静态文件在dist目录下
  
git：  
1. git init
2. git add .
3. git commit -m "..."
4. git remote add origin git@....git  
5. git push -u origin master
